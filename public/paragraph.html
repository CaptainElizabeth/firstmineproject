Paragraphs: How to write them

(Disclaimer: Should not be used in any English class)

So, you came to learn about Paragraphs, eh? Well, oh wise one, you're looking at it!

That's right! This text is a paragraph! Crazy, right?

The easiest way to write one is like so:

<p>Hello World!</p>

That comes out to, as expected: Hello World!

If you want to include < and > signs, then you have to enter in HTML Exit Keys

These Keys notify the HTML interpreter that the signs are NOT going to be tags

If <p> was put into a paragraph, the interpreter would mistakingly create another paragraph!

To combat this, entering "<" to replace <, and ">" to replace > will show the signs on the browser.

There is also another tag that breaks apart paragraphs, so it is not a long one (like this)

EXAMPLE:

<p>Hel<br>lo World<br>!</p>

That comes out to:

Hel
lo World
!

Keep in mind, if you're trying to make them double spaced, the browser automatically makes it single spaced. This is also true for spaces.
