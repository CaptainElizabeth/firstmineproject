The Early Text

HELLO MATEY! IT IS ME, CAPTAINELIZABETH! I AM TYPING IN BOLD TEXT!

That is right, Ladies and Gentlemen! There are multiple ways to write text in HTML!

Creating different types of text is very simple! Here is an example of Bold text:

<p><b>Hello World!<b><p>

That would come out as:

Hello World!

Now there are other types of text, which only require you to replace <b> from the previous example with another tag. Here are some of many examples:

This is emphasized Text. You use the tag <em>

This is italicized text. Use <i>

This text is smaller, if you didn't realize. Use <small>

Look at this:This is subscripted Text. Use <sub>

Look @ this: This is superscripted text. Use <sup>

THis is text as it appears in RubyMine. Use <code>. <kbd> also gets you this font

It's That simple!
