Tables and Table Rows

So, You've come to learn how to make tables, eh? Well, your in the right place!

Tables are grids that are made up of rows and columns

Have you ever been on Microsoft Excel? If you have (who hasn't), then you can probably guess what tables do

Lets see an example, shall we?

Mike Rotch 16
Oliver Clothesoff 23
Hugh Jass 19

That was an example of a table that had people's first and last name as well as their ages.

I know it doesn't LOOK like excel but its a table nonetheless

Now, the code is somewhat complicated, but I'll break it down for y'all:

<table border="1" style="width:50%"> <tr> <td>Mike</td>
<td>Rotch</td>
<td>16</td>
</tr>
<tr>
<td>Oliver</td>
<td>Clothesoff</td>
<td>23</td>
</tr>
<tr>
<td>Hugh</td>
<td>Jass</td>
<td>19</td>
</tr>
</table>

It may look complicated, but don't let looks decieve you

The <table style="width:100%"> defines how wide the table will be. In this case, the tables stretches accross half of the whole page (50% of the total width)

The <tr> tags stand for "table rows", and they create a new row in the table.

Also seen in the example is <td>. This stands for "table data" and inside it is where you should put all of your individual data. "Hugh" is a piece of data, so it goes in between the <td> tag.

Oh, I almost forgot! The border can be set as well!. The example is pretty self explanatory. (The border can be changed by changing the number).

And that's the basics on how to make tables!
