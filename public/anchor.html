Anchors

Welcome, Mateys! If you are wondering what Anchors are (The computer kind, not the physical one), you sir have just used one to get here!

Anchors, also known as Hyperlinks, allow the user to click on them and be transported to another web page or document. Cool, Right!

Now, there are many different types of hyperlinks, and they all are based on what "thing" the client clicks on to be redirected.

Here are some examples. This first one is for simple word hyperlinks:

<p><a href="anchor.html">Click Here</a> to Learn more about Anchors!</p>

Now, another example is for image hyperlinks, which allow the client to click on an image:

<a href="index.html"> <img src="fakeimage.gif" alt="Example Link" style="width:50px;height:50px;border:0"></a>

The example sets the image (fakeimage.jpg) to redirect to a certain document or website (index.html), and set the height and width of the image. (as well as the borders).

Basically,

<a>href=_" defines what the new website and/or document the client will be sent to

The text inside the <a> will be what text is hyperlinked.
